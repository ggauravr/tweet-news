About

Hash-Around was created in 24 hours at OSU Hackathon Fall '13, in a team of four. It's a Twitter and G-Maps API mashup web-application, to give location-based tweets about(using hashtags of the place) and around a place(using geolocation of the tweets), with an intuitive interface to navigate around places.

Instructions

  • Clone the repo in your local filesystem.. 
    $ git clone git@github.com:ggauravr/tweet-news.git
  • Fetch from remote(origin), the remote branch(gh-pages in left hand side) to the local branch(gh-pages in the right hand side).. 
    $ git fetch origin gh-pages:gh-pages
  • Checkout the newly created/updated branch.. 
    $ git checkout gh-pages
